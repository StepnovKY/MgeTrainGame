# 🚂 MGE TRAIN: SOSUNOK CHASE (C# Console Edition)

Добро пожаловать на арену, боец! Это не просто змейка, это **MGE TRAIN** — консольный симулятор доминирования, где ты управляешь составом **М-Г-Е**, преследуя прыгающих «сосунков».

## 🎮 Геймплей
Ты — паровоз **«М»**. Твоя цель — собирать «сосунков» (**+**), чтобы наращивать состав (**Г-Е-Е...**). 
Но будь осторожен: «сосунки» не стоят на месте, они постоянно прыгают, пытаясь уйти от твоего превосходства.

### Особенности:
- **Динамический состав**: Твой поезд буквально пишет «МГЕ» по мере роста.
- **Прыгающие цели**: Сосунки имеют шанс переместиться в случайную сторону каждый кадр.
- **Мемные фразы**: После проигрыша тебя поприветствуют классические цитаты из MGE TF2.
- **Управление**: 
  - `Стрелочки` — движение.
  - `ESC` — немедленный выход (для сопличков).
  - `R` — рестарт на экране смерти.

## 🛠 Установка и запуск

1. Убедись, что у тебя установлен **.NET SDK**.
2. Клонируй репозиторий:
    git clone https://github.com
3. Перейди в директорию проекта:
    cd MgeTrainGame
4. Перейди в папку и запусти проект:
    dotnet run
